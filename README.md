## 💡 Giới thiệu về Uxie
**Uxie** là một công cụ mã nguồn mở mạnh mẽ, giúp chuyển đổi các tài liệu tĩnh như PDF hoặc Markdown thành hệ thống học tập tương tác. Dự án tập trung vào việc sử dụng **AI Agent** để phân tích và sinh nội dung ngay trên môi trường Web, giúp bạn làm chủ kiến thức một cách thông minh hơn.

## 🚀 Triết lý Local-first trên Web
Uxie không sử dụng server-side để lưu trữ dữ liệu cá nhân, đảm bảo quyền riêng tư tối đa cho người dùng:
*   **Quyền riêng tư tuyệt đối:** Dữ liệu được mã hóa và lưu trữ duy nhất trên trình duyệt của người dùng, không cho phép bên thứ ba tương tác.
*   **Hiệu suất OPFS:** Sử dụng **Origin Private File System** – hệ thống tập tin ảo riêng biệt cho từng website, giúp truy cập dữ liệu với tốc độ cực cao ngay trên trình duyệt.
*   **Không phụ thuộc Server:** Toàn bộ quy trình xử lý từ lưu trữ đến tìm kiếm đều diễn ra cục bộ.

## ✨ Tính năng nổi bật
- [x] **Smart Rich-text Editor:** Trình soạn thảo dựa trên **Tiptap**, tối ưu cho việc ghi chú và biên tập tài liệu.
- [x] **AI-Powered Quiz:** Tự động tạo câu hỏi trắc nghiệm thông qua quy trình **RAG No-Embedding**, giúp tiết kiệm tài nguyên mà vẫn đảm bảo độ chính xác.
- [x] **Bento Grid Dashboard:** Giao diện Dashboard hiện đại, tinh gọn với phong cách thiết kế Pink & White năng động.
- [x] **Hybrid Search:** Kết hợp thuật toán **BM25** và tìm kiếm ngữ nghĩa để đảm bảo kết quả tra cứu tài liệu chuẩn xác nhất.

## 🛠 Công nghệ sử dụng (Web Stack)
*   **Framework:** Next.js & TypeScript.
*   **Styling:** Tailwind CSS với bố cục Bento Grid.
*   **Editor:** Tiptap Framework.
*   **Storage:** OPFS (Origin Private File System) & Browser Native Storage.
